# IronPulse Fitness

IronPulse Fitness is a modern, responsive gym landing page designed with a strong focus on clarity, discipline, and performance. The project demonstrates structured HTML, scalable CSS, and practical JavaScript for real-world frontend layouts.

# Features

* Fully responsive layout (desktop, tablet, mobile)

* Desktop-first design approach

* Mobile hamburger navigation menu

* Reusable section and card components

* Clean typography and consistent spacing

* CSS variables for theme management

* Subtle hover and interaction effects



# Tech Stack

* HTML5 (semantic markup)

* CSS3 (Flexbox, media queries, variables)

* JavaScript (vanilla)

* Font Awesome (icons)

* Google Fonts (Bebas Neue, Poppins)



# Responsiveness

The website is built using a desktop-first strategy and adapts to smaller screens using media queries.

Handled breakpoints:

* Desktop: 1024px and above

* Tablet: up to 768px
